# 校招岗位推荐平台

这是一个可外发的静态网站原型，包含：

- 产品介绍首页
- 首页、推荐岗位、岗位库、欢迎评论四个页面模块
- 推荐岗位区支持输入研究方向&研究领域进行岗位推荐
- 研究方向&研究领域支持多个关键词组合搜索，完整命中表格 B 列“岗位搜索关键词（人工）”的岗位优先，出现关键词但非完整命中的岗位排后
- 推荐岗位区支持按在招部门和地点筛选岗位
- 岗位库默认读取 171 个阿里星岗位
- 搜索在招岗位支持按岗位名称关键词实时搜索
- 欢迎评论区无需登录，评论会保存在当前浏览器中
- 推荐结果中提供可复制的岗位来源链接
- 每个推荐岗位提供“合适 / 不合适”人工反馈入口
- 推荐逻辑基于 `27届alistar岗位_岗位画像（最终版）.xlsx` 的全量岗位画像字段
- 已接入表格 B 列“岗位搜索关键词（人工）”，完整命中人工关键词优先排序，出现关键词的岗位排在后面，英文大小写不敏感
- 推荐结果精简展示岗位名称、在招业务、S/A/B/C 推荐等级、岗位链接和人工反馈
- 推荐数量可自定义选择 1-10，不选择时默认推荐 6 个
- 岗位库的岗位色块展示在招业务、地点、岗位大类和一级方向，并可点击跳转到对应岗位链接
- 推荐结果中的“复制链接”和“查看岗位详情”并排展示
- 已内置 `27届alistar岗位_岗位画像（最终版）.xlsx` 中的 171 个阿里星岗位画像
- 岗位库标题行右侧提供“导入岗位”入口，可临时替换 CSV 岗位库
- 页面视觉已调整为炫彩紫星光风格，并使用含星星的 Logo 样式

## 预览方式

直接打开 `index.html` 即可预览。

在推荐岗位区，输入研究方向&研究领域即可生成推荐岗位。

推荐算法会低权重处理 LLM、Python、PyTorch、C++、Linux、深度学习、分布式等泛化词；更重视 KV Cache、PagedAttention、vLLM、CUDA Kernel、TensorRT-LLM、Megatron、NCCL、SLAM、ROS、RISC-V、EDA、LLVM 等更能体现具体方向的信号。

## 当前岗位库

当前网站默认读取 `jobs-data.js`，其中包含 171 个岗位，来源于 `27届alistar岗位_岗位画像（最终版）.xlsx` 的“岗位画像总表”工作表。每个岗位保留岗位核心问题、岗位搜索关键词、必备能力、加分能力、泛化关键词、降权信号、HR 沟通提示等画像字段。

## CSV 岗位表建议字段

```csv
岗位名称,部门,地点,JD,必备要求,加分项
AI 应用工程师实习生,AI 平台部,上海,参与大模型应用和 RAG 建设,Python、RAG、LLM、FastAPI,LangChain、Agent、React
```

## 后续开发建议

第一版可以在此基础上接入后端数据库，保存岗位库、候选人信息和历史匹配结果；之后再增加 PDF 解析、OCR、账号权限和招聘系统对接。
